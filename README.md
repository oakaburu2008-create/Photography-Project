# Golden Hour Visuals

## Overview
**Golden Hour Visuals** is a simple photography portfolio website built using **HTML** and **CSS**. The website showcases a collection of photographs, highlights favorite images, provides a brief biography of the website, and includes contact information.

This project was created as a beginner web development project to practice HTML structure and CSS styling.

---

## Features

- Responsive webpage layout
- Header with website introduction
- Welcome (Hero) section
- Navigation menu
- Image gallery
- Favourite photos section
- Website biography
- Design description
- Contact information
- Footer with copyright

---

## Technologies Used

- HTML5
- CSS3

1. Download or clone this repository.

```bash

```

2. Open the project folder.

3. Double-click **index.html** or open it using any modern web browser such as:
   - Google Chrome
   - Microsoft Edge
   - Mozilla Firefox
   - Safari

## Website Sections

### Header
Introduces the website and its purpose.

### Hero Section
Welcomes visitors and explains the purpose of the photography website.

### Navigation
Contains links to:
- Home
- Gallery
- About
- Design

### Gallery
Displays a collection of photographs.

### Favourite Photos
Highlights selected favourite photographs with titles.

### Biography
Provides a short description of the website and its purpose.

### Design
Explains the overall design and layout of the website.

### Contact
Provides contact information including email and phone number.

### Footer
Displays copyright information.
